## 优先级选择器设计

**Why**: PRD 要求 v0.2.0 版本完善优先级设置功能,之前优先级只能在创建时设置为'medium',无法在UI中选择。

**What**:
- 设计了分段控制器 (Segmented Control) 样式的优先级选择器
- 三个按钮: 高(红色)/中(橙色)/低(绿色)
- 位于创建输入框和编辑模式中
- 选中状态有明显的视觉反馈(背景色+边框)
- 支持键盘导航(Tab切换,Enter选择,上下箭头切换)

**How to apply**:
- 创建任务时: 输入框和添加按钮之间
- 编辑任务时: 编辑输入框下方,按钮组上方
- 响应式: 移动端全宽度,桌面端自适应

## 搜索框设计

**Why**: PRD 要求 v0.2.0 版本新增任务搜索功能。

**What**:
- 顶部搜索框,支持实时过滤任务列表
- 包含搜索图标、输入框、清除按钮
- 200ms 防抖优化性能
- 支持键盘快捷键 (Ctrl/Cmd + F 聚焦,ESC 清空)
- 搜索无结果时显示友好提示

**How to apply**:
- 桌面端: 顶部栏中央,240px 宽度
- 移动端: 顶部栏下方独占一行,100% 宽度,高度 44px
- 布局变更: 移动端顶部栏高度从 56px 增加到 100px

## 交互优化

**What**:
- 优先级切换动画 (150ms 渐变 + 缩放效果)
- 搜索过滤动画 (FLIP 技术实现平滑过渡)
- 键盘导航增强 (支持箭头键切换优先级)
