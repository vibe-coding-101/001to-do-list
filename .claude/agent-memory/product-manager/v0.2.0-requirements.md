## v0.2.0 版本需求

**版本背景**：
- v0.1.1 已完成核心功能：创建、编辑、删除、标记完成、过滤任务
- 优先级字段已存在于数据模型中，但UI无法设置
- 缺少搜索功能，任务多时查找困难

**核心功能**：
1. **优先级设置完善**：创建/编辑时可选择优先级（高/中/低），有视觉区分
2. **任务搜索功能**：按关键词实时搜索，与过滤功能可叠加
3. **用户体验优化**：流畅的交互动画和清晰的操作反馈

**关键约束**：
- 不包含任务排序（按优先级/时间）
- 不包含高级搜索（多条件）
- 保持 v0.1.1 所有功能正常工作
- 不增加首屏加载时间

**Why**: 用户反馈 v0.1.1 版本优先级无法设置，任务多时查找困难，需要改进交互体验。

**How to apply**: 后续开发 v0.2.0 功能时，优先级设置和搜索是 Must，动画优化是 Should，排序功能不在本期范围。
