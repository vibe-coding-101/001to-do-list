## v0.2.0 核心设计变更

### 新增功能
1. **优先级选择器**
   - 位置: 创建任务时（输入框右侧）、编辑任务时（输入框下方）
   - 样式: 分段控制器，三按钮布局（高/中/低）
   - 交互: 单击切换，150ms 渐变动画
   - 默认值: 中优先级

2. **搜索框**
   - 位置: 桌面端（顶部中央）、移动端（标题下方）
   - 尺寸: 桌面端 240px 宽度，移动端 100% 宽度
   - 功能: 实时过滤，200ms 防抖
   - 交互: 支持清除按钮、ESC 清空

### 布局调整
- **桌面端**:
  - 顶部栏保持 64px 高度
  - 搜索框位于标题和过滤器之间
  - 底部输入区从 72px 增加到 80px

- **移动端**:
  - 顶部栏高度从 56px 增加到 100px
  - 搜索框独占一行（44px 高度）
  - 优先级选择器全宽度，均匀分布

## 设计规范要点

### 优先级选择器
- **高优先级**: 红色系（#FEF2F2 背景，#DC2626 文字）
- **中优先级**: 橙色系（#FFFBEB 背景，#D97706 文字）
- **低优先级**: 绿色系（#ECFDF5 背景，#059669 文字）
- **默认选中**: 中优先级

### 搜索框
- **高度**: 36px（桌面端）/ 44px（移动端）
- **圆角**: 8px
- **聚焦态**: 2px 蓝色边框 + 外发光效果
- **清除按钮**: 有内容时显示

### 交互动画
- **优先级切换**: 150ms 渐变 + 缩放效果
- **搜索过滤**: FLIP 技术实现平滑过渡
- **编辑模式展开**: 200ms 高度扩展 + 按钮滑入
