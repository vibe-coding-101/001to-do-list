## Bug 优先级定义

- **P0 (阻塞上线)**: 核心功能缺陷,必须修复才能上线
  - 数据丢失
  - 功能无法使用
  - 安全漏洞
  - 严重的兼容性问题

- **P1 (建议修复)**: 重要功能缺陷,强烈建议修复
  - 功能异常但可绕过
  - 用户体验问题
  - 非核心功能缺失

- **P2 (后续优化)**: 次要问题,可以后续迭代优化
  - UI 细节问题
  - 非关键交互问题
  - 优化建议
