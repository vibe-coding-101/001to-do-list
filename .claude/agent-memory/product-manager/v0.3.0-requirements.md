v0.3.0 版本于 2026-04-01 启动，核心方向是界面美化升级。

**Why:** v0.2.0 功能完整但视觉停留在"功能性原型"阶段，配色平淡、缺乏品牌感、微交互粗糙、移动端体验割裂。用户主观满意度需要提升。

**How to apply:**
- v0.3.0 仅修改 CSS 变量和组件 `<style>` 部分，不改动 template/script
- 原规划的虚拟滚动（FE-301~FE-306）推迟至后续版本
- PRD 位于 `docs/product-manager/PRD-v0.3.0.md`
- 关键改进点：全局配色升级、Header 渐变+毛玻璃、任务卡片左侧优先级竖条、复选框圆形化+弹性动画、胶囊过滤器 Tab、空状态升级、移动端优化
- 约束：不引入新依赖、不引入深色模式、backdrop-filter 需降级方案、动画需适配 prefers-reduced-motion
