v0.4.0 版本于 2026-04-01 规划，聚焦 **性能优化 + 视觉收尾**，不新增核心功能。

**核心内容**：
- Must: 虚拟滚动列表（支持 10000+ 任务）、搜索/过滤性能优化
- Should: 滚动位置记忆、过滤器 Tab 滑块动画、优先级选择器滑块动画
- Could: 优先级指示小圆点、空状态 SVG 配色更新

**推迟到 v0.5.0**：深色模式、任务分类/标签、任务截止日期、数据导出/导入

**Why**: v0.3.0 原规划的性能优化被推迟到 v0.4.0，v0.3.0 只做了界面美化。性能问题（10000 条任务卡顿）是当前最大瓶颈。

**How to apply**: v0.4.0 相关讨论以性能为第一优先级，视觉收尾为辅。新功能（深色模式、标签等）一律推到 v0.5.0。
