# 待办事项列表应用 v1.1+ 设计项目

**项目状态**: v0.1.1 已完成，v1.1+ 功能设计中

**已完成功能** (v0.1.1):
- 创建任务
- 查看任务列表
- 标记完成/未完成
- 编辑任务
- 删除任务
- 数据持久化 (LocalStorage)
- 响应式设计

**本次设计任务** (v1.1+):
- P1 功能（优先实现）:
  - 任务优先级（高/中/低）
  - 任务过滤（全部/未完成/已完成）
- P2 功能（设计先行）:
  - 任务搜索（关键词搜索）
  - 任务分类/标签（多标签支持）
  - 任务截止日期（日期选择）
  - 深色模式（主题切换）

**技术栈**: Vue 3 + TypeScript + Vite

**设计风格**: 简洁直观，Material Design 风格，响应式

**设计交付物**:
1. 更新的 design-spec.md（包含所有新功能的设计规范）
2. 5 个新的 Excalidraw 线框图文件
3. 更新的用户流程图（包含新功能流程）

**设计决策记录**:
- 优先级标签位置: 任务卡片右侧
- 标签数量限制: 建议最多 5 个
- 截止日期: 仅日期，不含时间
- 深色模式: 手动切换开关（自动切换可选）
